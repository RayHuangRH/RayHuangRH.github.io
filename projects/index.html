Course Site Generator

Overview

Course Site Generator is a desktop application that automates the creation of course webpages by creating all necessary HTML, CSS, JavaScript and JSON files on export. In the GUI, the user can choose custom images for the website along with custom descriptions for different website categories.

Course Site Generator also implements an undo/redo system for all changes made within the GUI and saving/loading so work can be continued at a later date if needed. Support for multiple languages is also included within the application.
