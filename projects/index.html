Course Site Generator

Overview

Course Site Generator is a desktop application that automates the creation of course webpages by creating all necessary HTML, CSS, JavaScript and JSON files on export. In the GUI, the user can choose custom images for the website along with custom descriptions for different website categories.

Course Site Generator also implements an undo/redo system for all changes made within the GUI, and saving/loading so work can be continued at a later date if needed. Support for multiple languages is also included within the application.

Quick Calculate

Overview

A web extension created with JavaScript, CSS and HTML. This extension allows the user to evalulate mathematical expressions following mathematical rules from within the Chrome browser. The user can also highlight an expression from a webpage and evaluate it directly through the webpage.

For this extension, I implemented the Chrome API to store the last six expressions and solutions. Chrome notifications is also implemented to notify the user of the solution to the expression.

Earth Defender

Overview

An arcade style game created with Java and CSS. In this game, the player aims to destroy the asteroids heading towards the earth while catching the astronauts that appear. For each asteroid destroyed, the player earns a point while points are lost for astronauts killed. All art and animations within the game were drawn by hand.

ADFGVX Cipher

Overview

The ADFGVX cipher was originally used by the German Army to encrypt messages during WWI. The cipher relies on matrix transposition to encrypt and decrypt its messages. The cipher was named ADFGVX because the encrypted message would be comprised of the six letters.

This program is written in MIPS Assembly and converts plaintext to and from an encrypted message using the ADFGVX cipher. This program also accepts custom encryption keys for different encrypted strings.

The Awesome Game

Overview

This game is written using Java and CSS. The objective is for the player to last as long as possible against increasing waves of enemies. At the end of each wave, the player must defeat a boss in order to progress to the next level. In between each level, the player is also given the option to shop and buy upgrades for the character.

This game has multiple enemy types along with custom attack styles and animations. A score and coin system is also implemented so the player is able to buy upgrades from the shop. The user can also rebind the controls for the player by changing the settings in the main menu.
